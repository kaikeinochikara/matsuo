# 顧問塾 ビジネスモデル整理

## 全体構造

```
【フロント：講座収益】               【バックエンド：エージェント収益】
                                    
簿記2級コース（298,000円）  ─┐
資格を短期取得したい人向け    │
                             ├──→ 転職エージェント事業（全卒業生対象）
簿記1級コース（498,000円）  ─┤        ↓
より高いポジションを狙う人    │    企業への紹介料収入
                             │    （年収の30〜35%が相場）
税理士コース（898,000円）   ─┘
独立のみ（エージェント対象外）
```

---

## 収益構造

### フロント（講座）

| コース | 価格 | ターゲット | エージェント対象 |
|--------|------|------------|----------------|
| 簿記2級 | 298,000円 | 資格を短期で取りたい人 | ○ |
| 簿記1級 | 498,000円 | 高ポジション狙い | ○ |
| 税理士 | 898,000円 | 独立志向 | ○ |

### バックエンド（転職エージェント）

- 対象：全コース卒業生（2級・1級・税理士）
- 収益モデル：企業への人材紹介 → 紹介料（成功報酬型）
- 紹介料試算：
  - 2級卒業生（推定年収300〜400万円）→ 紹介料 **90〜140万円/人**
  - 1級卒業生（推定年収450〜600万円）→ 紹介料 **135〜210万円/人**
  - 税理士卒業生（推定年収600万円〜）→ 紹介料 **180万円〜/人**

---

## ビジネスモデルの強み・論点

### 強み
- 講座で資格取得 → 転職支援まで一気通貫
- 卒業生の質が担保されているため企業側への訴求力が高い
- 講座単価が高いため、エージェントは"上乗せ収益"として機能

### 論点・深掘りしたい点
- [ ] 転職エージェントの収益化タイミング（講座終了後すぐ？資格合格後？）
- [ ] 企業開拓はどう行うか（既存ネットワーク／求人媒体連携など）
- [ ] 税理士コース卒業生の独立支援を別マネタイズできないか（事務所紹介など）
- [ ] 2級→1級→税理士へのアップセル設計
